## Overview

Tas7e7 is a restaurant management system focused on daily operations: order capture, table management, kitchen coordination, and management reporting. It is designed for cafes and full-service restaurants that need a fast POS workflow with a reliable back office.

---

## Tech Stack

<div align="center">

![Stack](https://img.shields.io/badge/Stack-Node%20%2B%20Express%20%2B%20MongoDB-339933)
![API](https://img.shields.io/badge/API-REST%20JSON-0080ff)
![Security](https://img.shields.io/badge/Security-JWT%20%2B%20bcrypt-black)

</div>

- Backend (server)
  - Node.js + Express (REST API)
  - MongoDB + Mongoose
  - JWT auth + bcryptjs
  - dotenv, CORS, config
  - ESC/POS printing via escpos + escpos-usb
- Frontend (client)
  - Next.js + React
  - TanStack React Query + Axios
  - Tailwind CSS + Radix UI
  - React Hook Form + Zod
  - i18next + language detection
