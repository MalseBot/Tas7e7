## Overview

Tas7e7 is a restaurant management system focused on daily operations: order capture, table management, kitchen coordination, and management reporting. It is designed for cafes and full-service restaurants that need a fast POS workflow with a reliable back office.

---

## What It Delivers

<div align="center">

![Orders](https://img.shields.io/badge/Orders-Real--time%20Tracking-2ea44f)
![Tables](https://img.shields.io/badge/Tables-Floor%20Management-1f6feb)
![Menu](https://img.shields.io/badge/Menu-Product%20Catalog-6f42c1)
![Kitchen](https://img.shields.io/badge/Kitchen-Preparation%20Queue-red)
![Reports](https://img.shields.io/badge/Reports-Sales%20Insights-orange)

</div>

- Order creation and status tracking across front-of-house and kitchen
- Table assignment and service status control
- Menu and item management
- Kitchen Display System (KDS) with preparation workflow
- Low-stock alerts and inventory visibility
- Role-based access for admins, managers, cashiers, and cooks
